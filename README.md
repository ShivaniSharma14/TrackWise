# TrackWise

TrackWise is a Django REST Framework backend project to manage personal productivity and daily life tracking.

## Features Completed
- Custom user authentication with JWT
- Expense CRUD API
- Expense filtering and ordering
- Habit and habit log models
- Ownership-based access control

## In Progress
- Habit routes
- Habit log validation
- Logout endpoint
- Testing

## Tech Stack
- Python
- Django
- Django REST Framework
- SimpleJWT
- SQLite
- django-filter

## Setup
1. Clone the repo
2. Create virtual environment
3. Install requirements
4. Add `.env`
5. Run migrations
6. Start server
